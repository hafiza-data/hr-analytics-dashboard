# HR Analytics Dashboard

## About the Project
In this project, I analyzed employee data to understand attrition trends and workforce distribution. The goal was to identify patterns related to employee turnover and key factors affecting it.

## Tools Used
Power BI (Data modeling, DAX, Visualization)

## What I Did
- Imported and cleaned employee data from CSV file
- Created KPIs like total employees, attrition count, and attrition rate
- Analyzed attrition based on department, salary, and other factors
- Built an interactive dashboard to visualize employee insights
